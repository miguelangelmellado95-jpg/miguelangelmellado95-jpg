### 🧭 How I work

Three repos, three stacks, **zero imports between them.** Cross-project handoffs are files on disk: the content engine drops markdown into the devlog's drafts folder, the game feeds the engine a string topic, and the devlog publishes whatever lands.

Each project has its own Claude Code session with its own `CLAUDE.md`. Every session ends with notes in `DEVLOG-NOTES.md`, which the devlog reads to draft posts in the same voice.
